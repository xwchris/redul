# Redul
An simple and fast React alternative with hooks and fiber (Excluded class-component).

## Advantages
- same hooks api with react
- simple code to read
- written with typescript
- small size

## Hooks
- useState
- useReducer
- useEffect
- useCallback
- useMemo
- useRef
